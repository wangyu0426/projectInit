Modals

Just some blank space
to push the page content
down so it is visible
below the modal.

The modal can be specified with a template file, in this case test/test-modal-detail.html.

Scope variables can be passed to the modal and they will update on the main scope and vice versa.

The modal is called with new modalDetail(templateUrl, data, onCloseCallbackFunction)

The modal can be closed from the parent controller using modal.close().
