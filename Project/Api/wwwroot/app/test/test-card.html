This a sample card page. It uses the <mtk-card> and <mtk-card-row> directives.

 

Syntax

Just like <mtk-form>, we have a container, and individual rows inside the container. 
                <mtk-card>
                     <mtk-card-row label="Address">VALUE OF ROW</mtk-card-row>
                     <mtk-card-row label="Name">VALUE OF ROW</mtk-card-row>
                 </mtk-card>

All the HTML markup and CSS classes are handled in the directives.

If you need to create two columns of card values side-by-side, you should create two separate <mtk-card> elements and place them in separate columns. (See the Tenancy example on this page).
