Rows and columns

Use class row to make a row that contains columns. A row must be inside a container-fluid, which the section containing the whole page is.

Use col-xs-N inside a row to make a block N columns wide. There are 12 columns in a row.

col-xs-12 fills the whole width

Paragraphs and spacing

For a thin gap below something, use an empty paragraph <p></p>

For a bigger gap use a paragraph with one of the classes v-space-16, v-space-24, v-space-32, or v-space-48.

To put a space above an element, without adding an extra element, add one of the classes mgt-8 or (margin-top-8), mgt-12 (or margin-top-12), mgt-16 (or margin-top-16), mgt-24 (or margin-top-24) to the element.

Examples

Paragraphs have a 9 pixel gap below them. An empty paragraph will create a 9 pixel gap.

 

&nbsp; is "non-breaking space" - a space the browser will always render. So this means the paragraph will be high enough for one line of text.
