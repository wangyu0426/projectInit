Note on the word "primary"

We use the word "primary" to describe our main blue colour as "Primary Blue".

Bootstrap uses the word "primary" to denote the main button for a form (normally "Save", and associated elements like panels which can be given a "primary" state. Each page or form generally has one primary button. All class names with the word "primary" use it in the Bootstrap sense, except the colour class PrimaryBlue.

 

The four states are: info success warning error (aka danger )

Use the classes .text-info, .text-success, .text-warning, .text-error to colour text. These colours are slightly darker than the pure PM colours (as they are for text on white and tinted backgrounds). These classes should be used, most of the time.
